# innovatelocal.ai

Marketing site for Innovate Local. Next.js, static export, deployed via GitHub Pages.

## Local development

```bash
npm install
npm run dev      # http://localhost:8080
npm run build    # generates ./out/
```

## Deploy

Push to `main`. The [Pages workflow](.github/workflows/static.yml) builds the site and publishes the `out/` folder.
